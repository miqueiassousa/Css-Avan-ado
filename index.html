Exemplos Avançados de CSS

1. Variáveis CSS (Custom Properties)

As variáveis permitem reutilizar valores como cores, tamanhos e espaçamentos ao longo do CSS, facilitando manutenção e personalização.

2. Grid Layout

O CSS Grid é uma poderosa ferramenta para criar layouts bidimensionais complexos, oferecendo controle sobre linhas e colunas de forma flexível.

3. Flexbox

O Flexbox simplifica a criação de layouts unidimensionais, permitindo o alinhamento e distribuição automática de elementos dentro de um contêiner flexível.

4. Animações e Transições

As transições suavizam mudanças entre valores de propriedades, e as animações permitem criar sequências de movimento ou efeitos ao longo do tempo.

Hover para animação

5. Pseudo-elementos e Pseudo-classes

Pseudo-elementos como ::before, ::after e pseudo-classes como :hover, :nth-child() ajudam a estilizar elementos com base em estado ou na sua posição.

Texto com pseudo-elemento ::before.

6. Media Queries e Design Responsivo

Media queries permitem criar layouts adaptativos para diferentes tamanhos de tela, essencial para design responsivo.

Redimensione a tela para ver o layout se ajustar!

7. CSS Shapes

O recurso CSS Shapes permite criar layouts que seguem formas geométricas, como círculos ou polígonos, ao invés de retângulos tradicionais.

8. Clip-Path e Máscaras

O clip-path permite recortar áreas de um elemento usando formas geométricas, enquanto o mask pode aplicar efeitos de máscara para revelar partes de um elemento.

Elemento com clip-path

9. Filtros (Filter)

O filter permite aplicar efeitos visuais como desfoque, contraste e saturação diretamente em imagens ou outros elementos.

10. CSS Scroll Snap

O Scroll Snap oferece controle sobre o comportamento de rolagem, permitindo que o conteúdo "trave" em pontos específicos ao rolar.

11. CSS Multi-Colunas

A propriedade columns divide o conteúdo de um contêiner em múltiplas colunas, útil para layouts de texto, como jornais e revistas.

O Subgrid permite que um item de um grid herde a estrutura de colunas ou linhas do grid pai, facilitando layouts complexos.

13. CSS Contain (Containment)

O contain melhora a performance ao limitar o impacto de um elemento nos cálculos de layout de outros elementos.

CSS Houdini é uma API experimental que permite criar novas funcionalidades CSS personalizadas, como propriedades e layouts, utilizando JavaScript.

A propriedade backdrop-filter aplica efeitos visuais (como blur) ao fundo do elemento, geralmente usado para criar efeitos de vidro fosco.
